—— ECMA-262(ECMAScript) 定义了 JavaScript 语言核心部分的标准

##### 1. ES6 新特性
* 支持常量
* 块范围（Block Scope）
* 箭头函数（=>）
* 扩展参数处理
* 模板字面值（Template Literals）
* 扩展字面值（Extended Literals）
* 增强的对象属性
* 解构分配（De-structuring Assignment）
* 模块
* 类
* 迭代器
* 生成器
* 集合
* 各种类的新内置方法
* 承诺（Promises） - JavaScript 异步编程特性
