#### Python 一键安装全部依赖包
* requirements.txt 用来记录项目所有的依赖包和版本号
```bash
# 导出开发所用 python 依赖包列表
pip freeze >requirements.txt
# 安装依赖包
pip install -r requirements.txt
```
